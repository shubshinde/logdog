Displays Debug Error Log's Inside Preety Footer Drawer.

== Description ==
With the help of LogDog you can check your websites errors,warnings,debug logs inside preety footer drawer. It can help developers to debug faster. Comes with Day-Night Mode feature.

== Installation ==
Just download plugin > Upload plugin > Upload/Install plugin > Activate 
In the top wordpress menu you will find 'LogDog' link for opening debug log drawer.

== Frequently Asked Questions ==

1. Do we collect your information ?
No, We do not collect your information.

2. Will LogDog be free for lifetime?
Yes, It will be free for life time. We believe in opensource, so it will be free forever.

3. Which pages will be modified by LogDog?
No pages will be modified, Debug drawer will be displayed at footer only in WordPress admin area.
